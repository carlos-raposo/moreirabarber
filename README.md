# Moreira Barber - Portfolio Project

## Sobre o Projeto / About the Project

**PT:** Este é um projeto de portfólio que apresenta um site estático para uma barbearia fictícia chamada Moreira Barber. O site foi desenvolvido utilizando tecnologias front-end modernas e está mantido como demonstração de habilidades de desenvolvimento web.

**EN:** This is a portfolio project featuring a static website for a fictional barber shop called Moreira Barber. The website was developed using modern front-end technologies and is maintained as a demonstration of web development skills.

## Como Executar Localmente / How to Run Locally

1. Clone o repositório / Clone the repository:
```bash
git clone https://github.com/carlos-raposo/moreirabarber.git
```

2. Navegue até a pasta do projeto / Navigate to the project folder:
```bash
cd moreirabarber
```

3. Abra o arquivo `index.html` em seu navegador ou use um servidor local:
```bash
# Usando Python 3
python -m http.server 8000

# Usando Node.js (http-server)
npx http-server
```

4. Acesse no navegador / Access in browser:
```
http://localhost:8000
```
